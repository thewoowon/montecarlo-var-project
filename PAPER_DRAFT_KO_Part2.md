### 3.2 VaR 및 CVaR 추정

**위험가치 (VaR)**:
VaR_α = -quantile(L, 1-α)

여기서 L은 포트폴리오 손실 분포이며, α는 신뢰수준(본 연구에서 0.95)이다.

**조건부 위험가치 (CVaR)**:
CVaR_α = -E[L | L ≤ -VaR_α]

CVaR은 VaR을 초과하는 기대손실을 나타내며, 추가적인 꼬리 리스크 정보를 제공한다.

### 3.3 시뮬레이션 방법

**몬테카를로 (MC)**:
1. 의사난수를 사용하여 Z ~ N(0, I_d) 생성
2. Cholesky 분해 적용: L = chol(Σ)
3. 변환: X = μ + ZL^T
4. 포트폴리오 수익률 계산: R_p = X · w

**준몬테카를로 (QMC)**:
1. 저불일치 수열 생성:
   - **Sobol**: Scrambled Sobol 수열 (scramble=True)
   - **Halton**: Scrambled Halton 수열 (scramble=True)
2. 균등분포를 정규분포로 변환: Z = Φ^{-1}(U)
3. Cholesky 분해 적용 및 수익률 계산 (MC와 동일)

### 3.4 분산 감소 기법

**Antithetic Variates**:
각 난수 벡터 Z에 대해 쌍대 표본 -Z를 생성하여 음의 상관관계를 활용함으로써 분산을 감소시킨다.

**Control Variates**:
조정된 수익률: R_adj = R - β(R_control - E[R_control])

여기서 β = Cov(R, R_control) / Var(R_control)

포트폴리오 기댓값 μ^T w를 control variate로 사용한다.

### 3.5 백테스팅 프레임워크

**Rolling Window VaR**:
- 추정 윈도우: 252일 (1년)
- 1일씩 롤링
- MC/QMC로 n=10,000 시나리오를 사용하여 VaR 추정
- 다음 날 실현 수익률을 VaR 예측치와 비교

**위반 지표**:
I_t = 1 if R_t < VaR_t, else 0

**Kupiec 검정 (무조건부 커버리지)**:

LR_uc = -2[log L(p_0) - log L(p_1)]

여기서 p_0 = 1-α (기댓값), p_1 = n_violations/n (관측값)

귀무가설 H_0: p_0 = p_1, χ²(1) 분포를 따름

**Christoffersen 검정 (독립성)**:

위반이 군집화(비독립성)를 보이는지 검정한다.

LR_ind = -2[log L_1 - log L_2]

여기서 L_1은 독립성을 가정하고, L_2는 마르코프 종속성을 허용한다.

**조건부 커버리지 검정**:

LR_cc = LR_uc + LR_ind ~ χ²(2)
